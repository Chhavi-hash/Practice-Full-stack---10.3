# social-media-app-aws
A simple Social Media Web App built using HTML, CSS, and JavaScript with basic user authentication, post creation, and feed display. Designed for AWS deployment using EC2 or S3 with scalability and cloud integration. Demonstrates login, posting, and real-time interaction features.
